Coding Vocabulary:

  • Angle Brackets: < >
  • Element: any code surrounded by angle brackets
  • Tag: the part of an element that opens or closes a code action
  • Attribute: everything in an element other than the tag
  • HTML: Hyper Text Markup Language (structures content)
  • CSS: Cascading Style Sheets (formats content for style)
  • JS: JavaScript (empowers interactivity of content)
  • URL: the alphanumeric address (the website's name)
  • IP address: the strictly numeric address (the website's real address)
  • DNS: the Domain Name Service, which matches URLs to IPs
  • Selector: CSS code that formats a selection of HTML content
